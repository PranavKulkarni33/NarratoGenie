What is NarratoGenie.AI?

NarratoGenie.AI transforms your PDFs into captivating, AI-generated interactive videos. Learn, teach, and create engaging content faster than ever before!

How It Works

1. Upload

Upload your PDF or document in a few clicks.

2. Transform

Our AI processes your content and generates a dynamic video narrative.

3. Preview

Watch and download your interactive video, ready to use or share.

Powered By

  • Angular
  • Express
  • AWS Polly
  • AWS Transcribe
  • AWS S3
  • AWS EC2 for hosting server
  • AWS S3 for static website hosting
  • AWS Route 53 for DNS
  • Hugging Face for summarization

Limitations

Currently, the application processes only the first 500 words for summarization due to limited resources, and the videos are kept short to stay within the free tier limits of the services used. At this stage, NarratoGenie.AI is restricted to handling only PDF files, and other file types are not yet supported. Additionally, real-time editing or advanced customizations for the generated videos are unavailable.

If we get funded, we plan to utilize OpenAI APIs to summarize entire documents with higher accuracy, support more file types such as DOCX and PPTX, create longer and more detailed videos with advanced animations, add interactive features to enhance video engagement, and scale the platform for broader usage and better performance.
